# AutoLux-The-Luxury-Cars-Showcase
A clean, simple car showcase website built with HTML, CSS, and JavaScript. It includes pages for Home, About, Contact, and Models. The Models page lists luxury cars with key details like price (₹), seats, engine (cc), and fuel type (Petrol/Electric/Hybrid). Easy to read, fast to load, and simple to customize.

## Features
- Clean multi-page layout: Home, About, Contact, Models
- Compact car spec blocks: price, seats, engine size, fuel type
- Fast and framework-free (pure HTML/CSS/JS)
- Easy to edit and extend for portfolios or demos

## Project Structure
- index.html — Landing page and navigation
- about.html — Overview of the site/brand
- contact.html — Contact info or form placeholder
- models.html — Car listings and specs (e.g., Mercedes-AMG GT with detailed attributes)

## Getting Started
1. Clone or download the repository.
2. Open `index.html` in any modern browser.
3. Edit `about.html`, `contact.html`, and `models.html` to customize content.

## Customization
- Add more cars by appending entries in `models.html`.
- Link a CSS file to style cards, typography, and layout.
- Add images, model links, or badges like “Electric/Hybrid”.

## Use Cases
- Student/portfolio showcase
- Simple product catalog
- HTML structure practice and demos
